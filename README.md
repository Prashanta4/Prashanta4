### 🏗️ Featured Engineering Systems

#### 🩺 [Gallbladder Web Inference System](https://github.com/Prashanta4/gallbladder-web)
* **Core:** Deep Learning (CNN) + Node.js Backend + React Frontend
* Engineered an end-to-end medical image processing pipeline using PyTorch CNN backends.
* Optimized binary classification accuracy for gallbladder disease detection while wrapping model execution in an asynchronous, non-blocking Express API layer to maximize request handling.

#### 📦 [PharmaTrack](https://github.com/Prashanta4/PharmaTrack-main)
* **Core:** Go (Golang) + PostgreSQL + Redis
* Built a high-throughput pharmaceutical supply chain ledger utilizing Go's native concurrency primitives (`goroutines` and `channels`).
* Implemented Redis distributed locks to eliminate race conditions during bulk stock tracking, guaranteeing strict transactional integrity.

#### 🖥️ [Ordis Terminal](https://github.com/Prashanta4/Ordis-Terminal)
* **Core:** Node.js + WebSocket + Custom IPC Protocols
* Developed a lightweight, real-time server management dashboard and pseudo-terminal interface.
* Utilized persistent WebSocket layers to stream sub-100ms server health metrics and administrative command streams with minimal overhead.

#### 📊 [HJ Cash Flow App](https://github.com/Prashanta4/HJ-Cash-Flow-App)
* **Core:** React / TypeScript + Express + PostgreSQL (Complex Aggregations)
* Built a financial auditing platform designed to parse multi-currency cash flows.
* Wrote optimized SQL window functions and relational indexing strategies to process financial rollups instantly without bottlenecks.

#### 🏫 [URMS UI & Tee-Web](https://github.com/Prashanta4/) *(University Management & E-Commerce Core)*
* **Core:** Next.js + Tailwind CSS + State Optimization
* Built highly responsive client-facing interfaces designed to handle high-traffic data layouts (student portals and product lines) with optimized client-side state machine management.
